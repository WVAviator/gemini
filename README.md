# Gemini Pet Rescue

Gemini is an animal rescue organization located in Northwest Mississippi. They adopt some animals locally and transport others to rescues out-of-state.

The website for Gemini will showcase any animals they have available for adoption listed on the organizations Shelterluv account through the use of Shelterluv's API. The website will also include the ability to accept donations through Stripe. Any additional content and copy of the website will be configurable through the use of Sanity.io headless CMS.

The website for Gemini uses the following technologies:
- Next.js
- React
- Typescript
- Sanity.io
- Emotion
- Stripe
- Sendgrid
